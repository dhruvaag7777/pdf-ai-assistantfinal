# 🚀 PDF AI Assistant (NLP Project)

This project is a simple yet practical PDF Assistant built using Python.

## 📌 Features
- Extracts text from PDF files
- Answers questions based on the content
- Generates summaries automatically

## 🔧 Tech Stack
- Python
- PyPDF2
- NLTK

## 🧠 How it Works
The application reads a PDF file, processes the text using Natural Language Processing (NLP) techniques, and performs:
- Sentence tokenization
- Word tokenization
- Frequency-based summarization
- Keyword-based question answering

## 💡 Key Learnings
- PDF text extraction using Python
- NLP basics (tokenization, frequency analysis)
- Building interactive CLI applications
- Writing structured and error-safe code
